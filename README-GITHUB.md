# Finance Dashboard 🚀

A stunning, futuristic finance dashboard built with React, TypeScript, and CSS. Features an immersive space-themed design with glass morphism effects, animated gradients, and smooth interactions.

## ✨ Features

- 🌙 **Dark/Light Mode Toggle** - Beautiful theme switching with smooth transitions
- 🎭 **Role-Based Access** - Switch between Viewer and Admin roles
- 💰 **Financial Overview** - Real-time summary cards with animated metrics
- 📊 **Transaction Management** - Add, view, filter, and search transactions
- 🎨 **Glass Morphism Design** - Modern frosted glass effects
- ⚡ **Advanced Animations** - Glitch effects, floating particles, shimmer animations
- 📱 **Fully Responsive** - Works perfectly on all screen sizes

## 🛠️ Tech Stack

- **React 18** - Modern React with hooks
- **TypeScript** - Type-safe development
- **Vite** - Lightning-fast build tool
- **CSS3** - Advanced CSS with animations and glass morphism
- **Context API** - State management

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/yourusername/finance-dashboard.git
cd finance-dashboard

# Install dependencies
npm install

# Start development server
npm run dev

# Open your browser
# Navigate to http://localhost:5173
```

## 🎯 Key Features Explained

### 🎨 Unique Design Elements
- **Space Theme**: Deep dark background with animated floating particles
- **Glass Morphism**: Frosted glass effects on all components
- **Neon Glow**: Glowing text and elements with neon shadows
- **Gradient Animations**: Moving gradient borders and backgrounds

### ⚡ Interactive Features
- **Glitch Effect**: Animated title with cyberpunk-style glitch
- **Hover Transformations**: Cards lift and scale on hover
- **Light Sweep Effects**: Buttons have animated light sweeps
- **Staggered Animations**: Transactions slide in with delays

### 💼 Financial Features
- **Real-time Calculations**: Balance, income, expenses, and savings rate
- **Transaction Categories**: Organize spending by category
- **Search & Filter**: Find transactions quickly
- **Role Permissions**: Admin can add transactions, Viewer can only view

## 🎮 Usage

1. **Switch Roles**: Use the dropdown to switch between Viewer and Admin modes
2. **Toggle Theme**: Click the 🌙/☀️ button to switch between dark and light modes
3. **View Transactions**: Browse your transaction history with search and filters
4. **Add Transactions** (Admin only): Click "Add Transaction" to add new entries
5. **Explore Animations**: Hover over cards and buttons to see the effects
